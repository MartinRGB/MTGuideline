动画

“There isn’t one easing curve that works for absolutely everything. Crafting easing curves is a key component of designing motion.” 动画设计相对更依赖感觉，新的写作平台的 Web 动画本质上追求一种明快清爽的运动感，而曲线使用又有所收敛,动画时间不宜超过 500ms。
主要使用的曲线有 5 种: linear , ease , easeInOut , easeOutQuint, easeModal,
linearcubic-bezier(0, 0, 1.0, 1.0) 适用于循环动画，以及体积微小元素、短时间（200ms内）动画。
easecubic-bezier(.25, .1, .25, 1.0) 适用于透明度动画，时间控制在 300ms 以内，并且效果不突兀。
easeInoutcubic-bezier(.42, 0, .58, 1.0) 适用于 Hover 动画，时间控制在 300ms 以内，并且效果不突兀。
easeOutQuintcubic-bezier(.23, 1, .32, 1) 运动感明快清爽，具有品牌感，时间最好在 500ms 以上，否则过快
easeModalcubic-bezier(.17,0.84,.44,1) 适用于模态弹窗，时间为 300ms，所有弹窗效果都应采用此曲线。
